# دليل إضافة قوالب مخصصة

## كيفية إضافة قوالبك الخاصة

إذا كنت تريد إضافة قوالب مخصصة من تصميمك الخاص، اتبع الخطوات التالية:

### 1. فتح ملف القوالب

افتح ملف `src/templates.ts`

### 2. إضافة قالب جديد

أضف قالب جديد إلى مصفوفة `templates` باتباع هذا النموذج:

```typescript
{
  id: 16, // رقم تعريف فريد (استخدم الرقم التالي المتاح)
  name: 'اسم القالب', // اسم القالب الذي سيظهر للمستخدم
  category: 'elegant', // الفئة: 'elegant' | 'modern' | 'islamic' | 'festive'
  gradient: ['#لون1', '#لون2', '#لون3'], // ألوان التدرج (من 2-3 ألوان)
  pattern: 'radial', // نوع النمط (انظر الأنواع المتاحة أدناه)
  textColor: '#FFFFFF', // لون النص
  titleSize: '4xl', // حجم العنوان: '4xl' | '5xl'
  subtitleSize: 'xl', // حجم العنوان الفرعي: 'xl' | '2xl'
}
```

### 3. أنواع الأنماط المتاحة (pattern)

- `'radial'` - تدرج دائري
- `'diagonal'` - تدرج قطري
- `'wave'` - موجات
- `'circular'` - دوائر
- `'geometric'` - أشكال هندسية
- `'floral'` - زخارف زهرية
- `'islamic'` - زخارف إسلامية (هلال ومسجد)
- `'stars'` - نجوم
- `'confetti'` - كونفيتي
- `'balloons'` - بالونات
- `'sparkle'` - لمعان
- `'organic'` - أشكال عضوية
- `'bubbles'` - فقاعات
- `'crystalline'` - بلورات
- `'luxury'` - فخامة

### 4. الفئات المتاحة (category)

- `'elegant'` - أنيق وفخم
- `'modern'` - عصري وحديث
- `'islamic'` - إسلامي تقليدي
- `'festive'` - احتفالي ومبهج

### 5. نصائح لاختيار الألوان

استخدم أكواد الألوان بصيغة Hex (#RRGGBB):
- للقوالب الأنيقة: ذهبي (#FFD700)، بنفسجي (#7B1FA2)، أزرق داكن (#1E40AF)
- للقوالب العصرية: ألوان زاهية ومتباينة
- للقوالب الإسلامية: أزرق وذهبي (#3B82F6 + #FFD700)
- للقوالب الاحتفالية: ألوان مبهجة (أحمر، وردي، برتقالي)

### 6. مثال عملي

```typescript
{
  id: 16,
  name: 'نجوم الليل',
  category: 'elegant',
  gradient: ['#1a1a2e', '#16213e', '#0f3460'],
  pattern: 'stars',
  textColor: '#FFD700',
  titleSize: '5xl',
  subtitleSize: '2xl',
}
```

### 7. إضافة أنماط مخصصة

إذا كنت تريد إضافة نمط زخرفة جديد تماماً:

1. افتح ملف `src/components/CardPreview.tsx`
2. ابحث عن دالة `drawPattern()`
3. أضف حالة جديدة في `if-else` للنمط الخاص بك:

```typescript
else if (template.pattern === 'my-custom-pattern') {
  // أضف كود الرسم الخاص بك هنا باستخدام Canvas API
  ctx.beginPath();
  // ... كود الرسم
  ctx.stroke();
}
```

## استخدام صور كخلفية

إذا كنت تريد استخدام صور كخلفية بدلاً من التدرجات، يمكنك:

1. وضع الصور في مجلد `public/`
2. تعديل ملف `CardPreview.tsx` لاستخدام صورة الخلفية

## احفظ وأعد البناء

بعد إضافة القوالب الجديدة:
```bash
npm run build
```
